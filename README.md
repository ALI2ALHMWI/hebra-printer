# hebra-printer
# 🖨️ متجر الطابعات - مشروع تدريب Git & GitHub

موقع بسيط لعرض منتجات الطابعات باستخدام **HTML**, **CSS**, و **JavaScript**.  
تم إنشاء هذا المشروع بهدف تعلم أساسيات **Git** و **GitHub** عملياً من خلال العمل على مهام حقيقية في فريق تدريبي.

---

## 📦 متطلبات التشغيل
- متصفح حديث (مثل Chrome, Firefox, Edge).
- محرر أكواد (مفضل VS Code).
- Git (اختياري – مطلوب فقط إذا أردت الاستنساخ والمساهمة).

---

## 🚀 كيفية تشغيل المشروع محلياً (خطوات التثبيت)

### الطريقة الأولى:直接用 المتصفح (بدون Git)
1. اضغط على زر **Code** في صفحة المستودع الرئيسية.
2. اختر **Download ZIP**.
3. فك الضغط عن الملف.
4. افتح المجلد ثم انقر مرتين على ملف `index.html` لفتحه في المتصفح.

### الطريقة الثانية: استخدام Git (للمساهمة في التطوير)
```bash
# 1. استنساخ المستودع (أو الـ Fork الخاص بك)
git clone https://github.com/username/printer-products-showcase.git

# 2. انتقل إلى مجلد المشروع
cd printer-products-showcase

# 3. افتح الملف index.html في المتصفح
#    أو استخدم إضافة Live Server في VS Code
